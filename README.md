## 📌 Project Overview

**Travel NZ** is a UI/UX design project for a New Zealand travel booking website. The goal was to design a platform that helps users discover destinations, plan itineraries, book tours, and access travel services — all in one place.

The project covers the full UX design process: from user research and persona creation, through wireframing, to a high-fidelity interactive Figma prototype.

---

## 🎯 Problem Statement

Travellers planning a New Zealand trip often have to visit multiple websites to book flights, accommodation, tours, and rental vehicles separately. This project explores a unified platform that simplifies the planning and booking experience for different types of travellers.

---

## 👤 Target Users

Two user personas were developed to guide design decisions:

### Sherley Mathew — Developer, 25–34
- **Goal:** Plan stress-free family holidays around birthdays, anniversaries, and New Year
- **Challenge:** Having to manage bookings across multiple websites is frustrating and time-consuming
- **Preferred communication:** Email
- **Tech-savvy**, active on social media (Facebook, Instagram, LinkedIn)

### Ivan Young — Bus Operator, 55–64
- **Goal:** Save up and take the family (including grandchildren) on a meaningful trip
- **Challenge:** Lacks guidance on finding family-friendly destinations
- **Preferred communication:** Phone, text, face-to-face
- **Less digitally experienced** — needs a simple, clear interface

## 🎨 Design Highlights

- **Brand identity:** Orange and dark circular logo with a luggage icon — warm, adventurous, approachable
- **Hero imagery:** Full-width New Zealand landscape photography (mountains, glaciers, coastline) on every page
- **Navigation:** Persistent top nav (Home / Tour / Contact) with active state highlighting in blue
- **Secondary nav:** Category filters below the hero for quick access to services
- **Accessibility considerations:** High contrast text, clear call-to-action buttons ("Book Now")
- **Support features:** Integrated chatbot and feedback form on the home page

## 🧠 Key UX Decisions

- **Unified booking platform** — addresses the core pain point of juggling multiple sites
- **Chatbot on home page** — supports less tech-confident users like Ivan without requiring them to navigate away
- **Itinerary planning feature** — allows users to view and customise a trip before committing to a booking
- **Social media integration** — caters to Sherley's social-savvy profile and provides trust signals for all users
- **Emergency contact visibility** — "Dial 111 in an Emergency" shown on the Contact page, relevant for international tourists
